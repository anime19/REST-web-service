### A simple web-service using REST.
<br><br>

### Functionalities ->
### 1) Retrieve all users
### 2) Create a user
### 3) Retrieve one user
### 4) Delete a user
### 5) Retrieve all posts from users
### 6) Create posts for a User
### 7) Retrieve details of a post.

<br><br>

### -> Spring Boot

### -> SQL
